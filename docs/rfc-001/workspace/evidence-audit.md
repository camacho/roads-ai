## Verification standard

A claim is verified only when all four checks pass:

1. **Existence:** the cited resource resolves and is the named work, source revision, or artifact.
2. **Support:** the cited locator directly supports the RFC's exact factual wording.
3. **Use:** the RFC preserves the source's population, intervention, comparison, outcome, and limitations.
4. **Recheck:** another reviewer can repeat the check from the locator or command recorded below.

Verdicts: `verified`, `qualified`, `contradicted`, `unavailable`.
